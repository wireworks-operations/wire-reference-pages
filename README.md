Wire Reference Pages

A collection of HTML-based reference pages dedicated to wire types, sizes, and related measurements. These resources are designed for quick lookup and operational support in wire handling and installation.

Overview
This repository contains static HTML files that serve as quick reference guides for various wire specifications, including:

Cable sizes (ACWU90, TECK90, Bare)
Cable diameter guides
Tape measure fractional reference

These pages are intended for offline or local use by operations teams, apprentices, and anyone needing fast access to wire data without relying on external tools.

Contents

ACWU90-Cable-Sizes.html – ACWU90 cable size reference
BARE-Cable-Sizes.html – Bare cable size reference
Cable-Diameter-Guide.html & Cable-Diameter-Guide-FINAL.html – Cable diameter charts
TECK90-1KV-Cable-Sizes.html – TECK90 1KV cable size reference
TECK90-600v-Cable-Sizes.html – TECK90 600V cable size reference
Tape-Measure-Fractional-Refrence.html – Fractional tape measure guide

Wire sizing and measurement references are critical for accurate cuts, installations, and compliance. This repo centralizes these resources for fast, reliable access without searching through multiple sources.
